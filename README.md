# Hotel Booking Backend API

Backend API for a hotel booking platform built using Node.js, Express, and MySQL.

## Features
- User Registration and Login (JWT Authentication)
- Search available rooms
- Book hotel rooms
- Secure endpoints with authentication

## Tech Stack
- Node.js
- Express.js
- MySQL
- JWT
- bcrypt

## Run Project

npm install
node server.js

Server runs on:
http://localhost:3000
